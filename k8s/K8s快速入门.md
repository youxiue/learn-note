1. 环境准备

关闭CentOs防火墙

```shell
#关闭CentOs防火墙
systemctl disable firewalld
systemctl stop firewalld

#安装etcd和kubernetes软件
yum install -y etcd kubernetes

#启动服务
systemctl start etcd
systemctl 
```
